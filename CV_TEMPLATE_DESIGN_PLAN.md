# CV Template Design Plan - Unique Layouts for Each Template

## Overview
This document outlines the unique design specifications for each CV template. Each template will have a distinct layout, color scheme, typography, and visual style to provide users with truly different options for their CVs.

## 1. MODERN Template ðŸŽ¨
**Design Philosophy:** Clean, contemporary, vibrant with gradient accents

### Layout Structure:
- **Header:** Full-width gradient background (blue-600 to blue-700), centered content
- **Profile Picture:** Circular, 80-100px, top-left of header with white border
- **Layout:** Two-column layout (30% sidebar, 70% main content)

  - Body: 11pt, gray-700
- **Unique Features:**
  - Gradient header with white text
  - Left sidebar with colored section backgrounds
  - Blue accent borders on experience items
  - Skill tags with blue background
  - Rounded corners on profile picture

### Section Order:
1. Header (with profile picture)
2. Summary (full width)
3. Two-column layout:

   - Right: Experience, Education, Projects

## 2. PROFESSIONAL Template ðŸ’¼
**Design Philosophy:** Corporate, formal, traditional, ATS-friendly

### Layout Structure:
- **Header:** Top border (4px gray-800), left-aligned name, contact info in single line
- **Profile Picture:** Rounded rectangle, 100-120px, right-aligned in header
- **Layout:** Single column, full width
- **Sections:** Stacked vertically with clear separation

### Visual Elements:
- **Colors:**
  - Primary: Dark gray (#1a202c) for text and borders
  - Accent: Gray-800 for section borders
  - Background: Pure white
- **Typography:**
  - Name: Uppercase, bold, 28pt, letter-spacing 1px
  - Section Titles: Uppercase, bold, 14pt, letter-spacing 2px, gray border bottom
  - Body: 11pt, serif font (Georgia), gray-700
- **Unique Features:**
  - Uppercase section titles with letter spacing
  - Thin horizontal dividers between sections
  - No colors except grayscale
  - Traditional serif font for body text
  - Formal date formatting
  - Minimal use of bullets (prefer paragraphs)

### Section Order:
1. Header (with profile picture right-aligned)
2. Professional Summary
3. Professional Experience
4. Education
5. Skills (horizontal list)
6. Certifications
7. Languages
8. Projects (if applicable)

**Design Philosophy:** Ultra-clean, maximum whitespace, simple lines

### Layout Structure:
- **Header:** Centered, minimal, no background color
- **Profile Picture:** Small circle (60-70px), subtle placement, top-center
- **Layout:** Single column, centered, generous margins

- **Unique Features:**
  - Maximum whitespace (40-50px margins)
  - Thin horizontal lines (1px) for section dividers
  - No background colors
  - Minimal use of bullets (dashes instead)
  - Centered alignment
  - Light font weights

### Section Order:
1. Header (centered, minimal)
2. Summary (centered, italic)
3. Experience (minimal spacing)
4. Education
5. Skills (simple comma-separated list)
6. Languages
7. Links (subtle)

## Implementation Strategy

### Phase 1: PDF Templates (React-PDF)
- Update each template's style file with unique colors, fonts, and layouts
- Implement unique layout structures for each template
- Add template-specific visual elements

### Phase 2: Live Preview Components
- Update `LivePreview.tsx` to render each template uniquely
- Match the PDF template designs in the web preview
- Ensure consistency between preview and PDF

### Phase 3: Template Preview Cards
- Update `TemplatePreview.tsx` to show accurate mini previews
- Each preview should reflect the actual template design
- Ensure users can see the difference before selecting

### Key Principles:
1. **Uniqueness:** Each template must be visually distinct
2. **Consistency:** Preview, PDF, and card previews must match
3. **Scalability:** Easy to add new templates following the pattern
4. **User Experience:** Clear visual differences help users choose
5. **ATS Compatibility:** Professional template remains ATS-friendly
